# BooksCollector — автотесты

# В проекте реализованы автотесты для класса BooksCollector
# Тесты написаны с использованием pytest и проверяют работу всех методов класса

# Что проверяется:

# 1. add_new_book
# Книга успешно добавляется
# Книга не добавляется, если длина имени > 40 символов

# 2. set_book_genre
# Жанр устанавливается корректно (валидный жанр)
# Жанр не устанавливается, если жанр не из списка

# 3. get_book_genre
# Возвращает правильный жанр книги

# 4. get_books_with_specific_genre
# Возвращает список книг с нужным жанром
# Если книг нет — возвращается пустой список

# 5. get_books_genre
# Возвращает словарь книг и жанров

# 6. get_books_for_children
# Возвращает только книги без возрастного рейтинга
# Книги с жанрами 18+ не попадают в список

# 7. add_book_in_favorites
# Книга добавляется в избранное
# Нельзя добавить книгу, если её нет в коллекции

# 8. delete_book_from_favorites
# Книга удаляется из избранного

# 9. get_list_of_favorites_books
# Возвращает список избранных книг
